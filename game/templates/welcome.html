What.is.1000?

This excellent three-player game is well known in several Eastern European countries, including Russia, Belorus, Ukraine, Kazakhstan, Poland and Lithuania. For more information please read the rules.

About.the.project

This project was started as part of Python course at my university. I needed the topic for the project so I have chosen this game because I know and like it very much. My friend and I play this game very often during breaks and lectures. We find it very addictive. So I realized that it would be great idea to be able to play it online. This way we could play anywhere! And finally I could deliver this game to masses of people.
